# Workspace Monorepo

This is a consolidated monorepo containing all projects from the workspace.

## Structure

- **DomainController/** - Domain controller scripts and configurations
- **MQL5-Google-Onedrive/** - MQL5 Google Onedrive integration
- **OS-Twin/** - OS-Twin project
- **OS-Twin-setup/** - OS-Twin setup scripts
- **AgentBrain/** - AgentBrain project

## Repository History

This monorepo was created by merging multiple repositories while preserving their individual git histories where possible.

## Usage

Each subdirectory maintains its original structure and can be worked on independently.
